# Casos de Uso

Colección de casos de uso reales para cada tipo de colección.

## Índice de casos

### List
- **ArrayList**: Catálogos de productos, datos de sensores, listas de reproducción
- **LinkedList**: Historial de navegación, undo/redo, colas de transacciones

### Set
- **HashSet**: IPs bloqueadas, tags únicos, cupones usados
- **TreeSet**: Eventos de calendario, top N puntuaciones, análisis de temperaturas
- **LinkedHashSet**: Mantener orden de inserción con unicidad

### Map
- **HashMap**: Caché de usuarios, carrito de compras, contadores de frecuencia
- **TreeMap**: Horarios ordenados por tiempo, historial de precios, versionado de documentos
- **LinkedHashMap**: Caché LRU, configuración ordenada

### Queue
- **PriorityQueue**: Triage hospitalario, colas de impresión, algoritmos de pathfinding
- **ArrayDeque**: Stack de navegación, evaluación de expresiones, BFS/DFS
