# Streams API

Programación funcional con colecciones en Java.

## Características principales

- **Pipeline**: Operaciones encadenadas
- **Lazy evaluation**: Evaluación perezosa
- **Paralelismo**: parallelStream()

## Operaciones intermedias

- `filter()`: Filtrar elementos
- `map()`: Transformar elementos
- `flatMap()`: Aplanar estructuras anidadas
- `sorted()`: Ordenar
- `distinct()`: Eliminar duplicados

## Operaciones terminales

- `collect()`: Recolectar en colección
- `reduce()`: Reducir a un valor
- `forEach()`: Iterar
- `count()`: Contar elementos
