# Algoritmos con Collections

Métodos de utilidad en la clase Collections.

## Ordenamiento

- **sort()**: Ordena una lista
- **reverse()**: Invierte el orden
- **shuffle()**: Orden aleatorio
- **rotate()**: Rota elementos

## Búsqueda

- **binarySearch()**: Búsqueda binaria O(log n)
- **max() / min()**: Elementos extremos
- **frequency()**: Contar ocurrencias

## Modificación

- **fill()**: Llenar con valor
- **copy()**: Copiar elementos
- **swap()**: Intercambiar posiciones
- **replaceAll()**: Reemplazar valores
